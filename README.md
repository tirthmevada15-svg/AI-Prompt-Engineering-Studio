# ✨ AI Prompt Engineering Studio

A modern **AI Prompt Engineering Toolkit** built with **Python, Streamlit, and Llama models**.
This application allows users to **generate, improve, analyze, and manage AI prompts** through a clean **dark neon dashboard interface**.

Designed as an **AI productivity tool** for creators, developers, and prompt engineers.

---

# 🚀 Features

### 🪄 Magic Prompt Generator

Convert a simple **title or idea** into a powerful and structured AI prompt.

Example input:

`YouTube video about AI tools for students`

The system generates a detailed prompt including:

* Role
* Context
* Instructions
* Tone
* Output format

---

### ⚡ Prompt Improver

Enhance an existing prompt to make it:

* clearer
* more detailed
* structured
* optimized for AI models

---

### 📊 Prompt Quality Analyzer

Evaluate prompt quality with AI.

Returns:

* Prompt Quality Score
* Strengths
* Weaknesses
* Suggestions for improvement

---

### 📚 Prompt Template Library

A collection of ready-to-use prompts across categories:

* Marketing
* Content Creation
* Coding
* Business

These templates help users quickly start writing better prompts.

---

### 📄 Download Prompt as PDF

Generated prompts can be exported and downloaded as **PDF files** for later use.

---

### 🎨 Dark Neon Dashboard UI

The application uses a modern **glassmorphism + neon style dashboard** built with Streamlit.

Features include:

* Dark futuristic theme
* Neon glow accents
* Glass card UI components
* Clean sidebar navigation

# ⚙️ Installation

Clone the repository:

```
git clone https://github.com/yourusername/ai-prompt-engineering-studio.git
```

Move into the project directory:

```
cd ai-prompt-engineering-studio
```

Install dependencies:

```
pip install -r requirements.txt
```

---

# 🤖 Install Llama (Ollama)

Download and install Ollama:

https://ollama.com

Pull the Llama model:

```
ollama pull llama3
```

Run the model locally:

```
ollama run llama3
```

---

# ▶️ Run the Application

Start the Streamlit app:

```
streamlit run app.py
```

Open the browser:

```
http://localhost:8501
```

---

# 🌐 Deploy on Streamlit Cloud

1. Push the project to **GitHub**
2. Open **Streamlit Community Cloud**
3. Create a **New App**
4. Select the repository
5. Choose `app.py` as the main file
6. Deploy

Your application will be live on the web.
